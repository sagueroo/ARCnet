# ARCnet
Une solution d'automatisation réseau basée sur l'intention (Intent-Based Networking) visant à automatiser le provisionnement de services BGP/MPLS VPN. L'outil transforme un modèle de données centralisé (JSON) en configurations déployables, assurant le routage du cœur de réseau et l'isolation sécurisée des clients via des VRF.
